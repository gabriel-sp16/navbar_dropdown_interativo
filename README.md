# navbar_dropdown_interativo
Barra de navegação (Navbar) moderna com menu dropdown integrado e animações fluidas. Ao passar o cursor pelo menu, o componente revela um submenu suspenso estilizado com sombreamento leve, transições suaves de opacidade e rotação interativa de ícones indicadores do Font Awesome, além de incluir um botão estruturado para chamadas de ação.
